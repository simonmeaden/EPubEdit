Detailed Description

The ExtendedColorDialog class provides an extension of the standard QColorDialog class that includes selection pages for both the SVG color name set and the X11 color name set.

The dialog allows the user to display two different color fields with a text color so that thet can be compared. Colors can be recovered using the color() and textColor() methods and initially set using the setColor() and setTextColor() methods. The dialog allows the user to switch between a standard color selection page and various SVG and X11 color sslection pages. Clicking on a color will set the primary (left) display. Alternatively the user can drag colors to either left or right display, or use the context menu to set the color, or text color of either.
