◆ addItem()

WidgetItem* AbstractDockItem::addItem ( WidgetItem *  item)
protected

Adds a prebuilt widget to the DockToolbar/DockHeader/DockFooter.

This is used internally to allow the user to clone the appropriate DockItem. This was originally added to allow the MultiSplitter widget to clone it's various widgets along with their headers and footers without having to totally rebuild them from scatch.

WidgetItem* AbstractDockItem::takeAt ( int  index)
