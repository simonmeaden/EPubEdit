Detailed Description

Creates an interface that allows your QObject extensions to clone themselves.

Intended for use by the MultiSplitter widget. When the splitter split's a QWidget it checks the existing widget, and if it extends the CloneableObjectInterface class it will be cloned, placing a copy of the widget in each split. Otherwise it will attempt to create a copy of the QWidget.
