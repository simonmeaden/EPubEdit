Detailed Description

Scanner/Parser for html code.

Walks throughan html page, detecting and marking html tags and detecting incorrect code, such as misspelled tag and attribute names.

See also
HtmlHighlighter

◆ matchedTag()

Tag* HtmlScanner::matchedTag ( Tag *  tag)

Returns the next matching tag to the supplied tag.

The match returned is the matching tag of the same type. For example if a

<html ..>

tag is supplied then the first matching

<\html>

will be returned. Conversely if a

</html>

is supplied the first previous

<html ...>

will be returned. If there is no matching tag then

nullptr

is returned.
