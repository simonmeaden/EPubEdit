◆ setWidget()

QWidget* AbstractDockWidget::setWidget ( QWidget *  widget)

Sets the central widget of the DockWidget.

If the new QWidget is replacing an existing widget a pointer tothe old QWidget is returned, otherwise a nullptr is returned. The old QWidget is NOT deleted.

Note
If the QWidget is replaced with itself, the pointer is returned but nothing else is done. You will need to delete it yourself.
