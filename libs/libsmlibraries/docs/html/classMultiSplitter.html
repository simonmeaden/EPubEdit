Detailed Description

A multidirectional splitter that works in a similar way to the editor panes in QtCreator.

The MultiSplitter is actaully an array of QWidget's and QSplitters. Initially it holds a single QWidget. When the widget is split it replaces the widget with a QSplitter, and places the QWidget and a second QWidget.

Classes that intend to be used in MultiSplitter should implement the CloneableWidgetInterface. This creates a cloneable widget so that when a widget is 'split' either vertically or horizontally a clone of the forst widget is created and placed into the second split.
