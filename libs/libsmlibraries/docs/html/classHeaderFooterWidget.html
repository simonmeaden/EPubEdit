Detailed Description

The HeaderWidget class defines a HeaderWidget object.

"includes/headerfooterwidget.h"

A HeaderFooterWidget is a QWidget with both a header a footer bar in which various widgets* can be placed. The actual widgets are stored in a DockHeader and a DockFooter which can be accessed via the header() and footer() methods respectively.
