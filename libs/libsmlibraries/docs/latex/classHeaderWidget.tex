\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classHeaderWidget}{Header\+Widget}} class defines a \mbox{\hyperlink{classHeaderWidget}{Header\+Widget}} object. 

A \mbox{\hyperlink{classHeaderWidget}{Header\+Widget}} is a QWidget with a footer bar in which various widgets can be placed. The actual widgets are stored in a \mbox{\hyperlink{classDockHeader}{Dock\+Header}} which can be accessed via the \mbox{\hyperlink{classHeaderWidget_a21a04bb444f9619a7c70520d53b82331}{header()}} method.
