\doxysubsection{Detailed Description}
The Labelled\+Widget abstract base class. 

\begin{DoxySince}{Since}
5.\+7.\+0 
\end{DoxySince}

The class is designed as a common super class for widgets like \mbox{\hyperlink{classLabelledSpinBox}{Labelled\+Spin\+Box}} and \mbox{\hyperlink{classLabelledComboBox}{Labelled\+Combo\+Box}}.

The label can be on either side of the combo box, above it or below, with the position being set using the Labelled\+Combo\+Box\+::set\+Position() method.

The label\textquotesingle{}s value can be set/recovered using the text() and set\+Text methods, and their format can be modified using text\+Format() and set\+Text\+Format().

The label has full alignment capability using \mbox{\hyperlink{classAbstractLabelledWidget_a77c44c5b838802bdc63eefd75196e161}{Labelled\+Combo\+Box\+::set\+Label\+Alignment()}}. Use the Labelled\+Combo\+Box\+::value() method to access.

Stylesheets can be set for the individual components using either \mbox{\hyperlink{classAbstractLabelledWidget_aa10a84728f9948a9e0f00d5ba65e03df}{set\+Label\+Style\+Sheet()}} or set\+Wrapper\+Style\+Sheet(). Alternatively you can set a combined stypesheet using set\+Style\+Sheet() of the wrapping QWidget, however stylesheets set for individual components will override those set for the wrapper.

QSize\+Size\+Policy can be set for either component using \mbox{\hyperlink{classAbstractLabelledWidget_a5f8a68b3cdaca7e00d58f969794eb690}{set\+Label\+Size\+Policy()}} or \mbox{\hyperlink{classAbstractLabelledWidget_aad69e3b423e19535f96ebdc735a90d63}{set\+Widget\+Size\+Policy()}}.
