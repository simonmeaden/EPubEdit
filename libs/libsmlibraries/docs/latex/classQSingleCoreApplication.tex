\doxysubsection{Detailed Description}
The QSingle\+Application class limits the application to a single instance. 

Extends QCore\+Application and checks for an already existing instance on the computer. If it detects one it closes the new attempt. To use just extend \mbox{\hyperlink{classQSingleCoreApplication}{QSingle\+Core\+Application}} rather than QCore\+Application.
