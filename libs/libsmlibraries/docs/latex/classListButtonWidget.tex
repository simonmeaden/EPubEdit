\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classListButtonWidget}{List\+Button\+Widget}} class supplies a button with a drop down menu. 

\mbox{\hyperlink{classListButtonWidget}{List\+Button\+Widget}} is a subclass of \mbox{\hyperlink{classButtonWidget}{Button\+Widget}} that supplies a drop down menu when it is clicked.

Preferably add\+Icon\+Text\+List\+Button(...) \mbox{\hyperlink{classDockHeader}{Dock\+Header}}, \mbox{\hyperlink{classDockFooter}{Dock\+Footer}} or \mbox{\hyperlink{classDockToolbar}{Dock\+Toolbar}} should be used to create the button as it assures the button is properly created.
