\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classListItem}{List\+Item}} class is used to store items in \mbox{\hyperlink{classListWidget}{List\+Widget}}\textquotesingle{}s and \mbox{\hyperlink{classListButtonWidget}{List\+Button\+Widget}}\textquotesingle{}s. 

When an item is added to \mbox{\hyperlink{classListButtonWidget}{List\+Button\+Widget}} then a List\+Item$\ast$ is returned. This can then be used to connect the \mbox{\hyperlink{classListItem_a60f208e9a93cdf208456744342127407}{List\+Item\+::item\+Clicked}} signal to your receiver method.

\mbox{\hyperlink{classListItem}{List\+Item}}\textquotesingle{}s are not returned when used in a \mbox{\hyperlink{classListWidget}{List\+Widget}} as instead the \mbox{\hyperlink{classListWidget_a2c269d52f72e00f18e6c158ce013a81b}{List\+Widget\+::item\+Clicked}} signal has parameters of the int index and QString text of the item.

You should never need to create one of these items manually as they are created internally when items are added to the \mbox{\hyperlink{classListButtonWidget}{List\+Button\+Widget}}.
