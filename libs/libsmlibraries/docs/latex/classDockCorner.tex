{\footnotesize\ttfamily Dock\+Corner\+::\+Dock\+Corner (\begin{DoxyParamCaption}\item[{Corner\+Type}]{type,  }\item[{Dock\+Position}]{position,  }\item[{\mbox{\hyperlink{classAbstractDockWidget}{Abstract\+Dock\+Widget}} $\ast$}]{parent }\end{DoxyParamCaption})}



Constructor for a \mbox{\hyperlink{classDockCorner}{Dock\+Corner}} specifying type, position and parent. 

Type has three useable values, Box, VEdge and HEdge. None can have unexpected results depending on addition order.


\begin{DoxyItemize}
\item Box which specifies a blank box at the corner.
\item VEdge which allow the North/\+South items to reach the edges. East/\+West items butt against the North/\+South items.
\item HEdge which allow the West/\+East items to reach the edges. North/\+South items butt against the East/\+West items.
\item None No corner is specified and the layout will depend on addition order.
\end{DoxyItemize}

Position specifies which corner. Dock\+Position actually has nine possible values, but only four Dock\+Position\+::\+North\+East, Dock\+Position\+::\+North\+West, Dock\+Position\+::\+South\+East and Dock\+Position\+::\+South\+West are valid for a \mbox{\hyperlink{classDockCorner}{Dock\+Corner}}. Using other values can have anomalous results.
