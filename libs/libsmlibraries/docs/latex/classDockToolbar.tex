\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classDockToolbar}{Dock\+Toolbar}} class defines a \mbox{\hyperlink{classDockToolbar}{Dock\+Toolbar}} object. 

\mbox{\hyperlink{classDockToolbar}{Dock\+Toolbar}}\textquotesingle{}s are similar to QToolbars in that you can add certain types of widget to them, primarily certain types of \mbox{\hyperlink{classButtonWidget}{Button\+Widget}}, a \mbox{\hyperlink{classLabelWidget}{Label\+Widget}} or a \mbox{\hyperlink{classListWidget}{List\+Widget}}.
