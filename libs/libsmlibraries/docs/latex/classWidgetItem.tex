\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classWidgetItem}{Widget\+Item}} class is the base class for all dock widget items. 

\mbox{\hyperlink{classWidgetItem}{Widget\+Item}} provide an abstract class for creation of \mbox{\hyperlink{classWidgetItem}{Widget\+Item}}\textquotesingle{}s.
