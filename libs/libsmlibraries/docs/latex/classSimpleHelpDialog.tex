\doxysubsection{Detailed Description}
An help window that displays a simple help dialog. 

The dialog simply displays the help string and a \textquotesingle{}Close\textquotesingle{} button. The \textquotesingle{}Close\textquotesingle{} button returns no value is returned so use the show() method rather that exec() method to invoke the dialog.
