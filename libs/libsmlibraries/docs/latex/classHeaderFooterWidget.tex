\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classHeaderWidget}{Header\+Widget}} class defines a \mbox{\hyperlink{classHeaderWidget}{Header\+Widget}} object. 

\char`\"{}includes/headerfooterwidget.\+h\char`\"{}

A \mbox{\hyperlink{classHeaderFooterWidget}{Header\+Footer\+Widget}} is a QWidget with both a header a footer bar in which various widgets$\ast$ can be placed. The actual widgets are stored in a \mbox{\hyperlink{classDockHeader}{Dock\+Header}} and a \mbox{\hyperlink{classDockFooter}{Dock\+Footer}} which can be accessed via the \mbox{\hyperlink{classHeaderFooterWidget_a189e9573301a0542d15a9ad94e6cded0}{header()}} and \mbox{\hyperlink{classHeaderFooterWidget_a78e9582fc8e4451e8040ab7431397255}{footer()}} methods respectively.
