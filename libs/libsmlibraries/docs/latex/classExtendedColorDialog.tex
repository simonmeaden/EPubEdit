\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classExtendedColorDialog}{Extended\+Color\+Dialog}} class provides an extension of the standard QColor\+Dialog class that includes selection pages for both the SVG color name set and the X11 color name set. 

The dialog allows the user to display two different color fields with a text color so that thet can be compared. Colors can be recovered using the \mbox{\hyperlink{classExtendedColorDialog_ade3ea5ef25c0cf54f405969c77ae8ae4}{color()}} and \mbox{\hyperlink{classExtendedColorDialog_aabc32e00733c429af0f2ec6d0d21f922}{text\+Color()}} methods and initially set using the \mbox{\hyperlink{classExtendedColorDialog_a671d108665b9db86c70c78279181dc99}{set\+Color()}} and \mbox{\hyperlink{classExtendedColorDialog_a246501d129e8e4d667b7bbd9daf211a6}{set\+Text\+Color()}} methods. The dialog allows the user to switch between a standard color selection page and various SVG and X11 color sslection pages. Clicking on a color will set the primary (left) display. Alternatively the user can drag colors to either left or right display, or use the context menu to set the color, or text color of either.
