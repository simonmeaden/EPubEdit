\doxysubsection{Detailed Description}
Creates an interface that allows your QObject extensions to clone themselves. 

Intended for use by the \mbox{\hyperlink{classMultiSplitter}{Multi\+Splitter}} widget. When the splitter split\textquotesingle{}s a QWidget it checks the existing widget, and if it extends the \mbox{\hyperlink{classCloneableObjectInterface}{Cloneable\+Object\+Interface}} class it will be cloned, placing a copy of the widget in each split. Otherwise it will attempt to create a copy of the QWidget.
