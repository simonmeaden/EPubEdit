\doxysubsection{Detailed Description}
A class that specifies information about a single language tag, extended language tag, script tag, regional tag or grandfathered tag. 

There are several different types of tag.
\begin{DoxyItemize}
\item LANGUAGE These are the primary language tags.
\begin{DoxyItemize}
\item A two character ISO639-\/1\+:2002 code.
\item A three character ISO 639-\/2\+:1998, ISO 639-\/3\+:2007 or ISO 639-\/5\+:2008 code.
\item A single character \textquotesingle{}i\textquotesingle{} or \textquotesingle{}x\textquotesingle{} or a three character code in the range \textquotesingle{}qaa\textquotesingle{} to \textquotesingle{}qtz\textquotesingle{} which are solely for non-\/registered private usage.
\end{DoxyItemize}
\item EXTLAN These are language extensions
\item SCRIPT These indicate the possible script for the language.
\item REGION These indicate the regional variation for the language.
\item VARIANT These indicate a variation on the primary language.
\item GRANDFATHERED These are generally older tag names which shouldn\textquotesingle{}t really be used any more. They generally have a prefered\+Value() that shows the tag that you should now use. This can be recovered using prefered\+Value().
\item REDUNDANT These are generally older tag names which shouldn\textquotesingle{}t really be used any more.
\end{DoxyItemize}

For a more extensive handling see \href{https://www.w3.org/International/articles/language-tags/}{\texttt{ https\+://www.\+w3.\+org/\+International/articles/language-\/tags/}}.

This class loads the language tag definitions from the \href{https://www.iana.org/assignments/language-subtag-registry/language-subtag-registry}{\texttt{ IAIN language tag registry}} and parses it into an easier to use-\/ish form. It will also save the data into a local YAML file for easier/faster recovery.
