\doxysubsection{Detailed Description}
A utility class to supply IAIN language tag names. 

The complete list of languages tags. These are stored in a set of QString=$>$\mbox{\hyperlink{classBCP47Language}{BCP47\+Language}} maps. The full list of mixed tag types can be recovered using the complete\+Map() method, alternatively you can use language\+Map(), extlan\+Map(), region\+Map(), script\+Map() or variant\+Map() to recover the specialised maps.

The data for the list is recovered directly from the IANA website and requires web access. The the list should not be manually created. The web connection is only required the first time, although if it is available the file-\/date is checked and if a newer file is available it will be updated accordingly. Once downloaded and parsed it is stored locally in a YAML file to enable faster access. Downloading and parsing are both done in seperate threads so as not to delay the main application.

Data is stored in a map of registry description against \mbox{\hyperlink{classBCP47Language}{BCP47\+Language}} objects.

Region names, defined by the \mbox{\hyperlink{classBCP47Language_af32a4c60911b4630fb9e7be7d9d68b04a4d4597330df0532b6cc4d33cf5435fd6}{BCP47\+Language\+::\+REGION}} type are used to extend a language in a similar way to the EXTLANG types. However these types ONLY define the regional code and do not specify the primary language.

Region names are used with a hyphon to specify a regional variation of a primary language for example Canadian French would be specified by fr-\/\+CA with fr being the primary language and CA specifying Canada as a region.

Prefixes are defined using the xx\+\_\+yyy format where xx is the main language defined by \mbox{\hyperlink{classBCP47Language_a0fcf76727c52931c67b094bf74057e1e}{BCP47\+Language\+::prefix()}}.at(0) and yyy is the sub-\/language defined by \mbox{\hyperlink{classBCP47Language_aaf4db218966cab0e9d104dfe806c6423}{BCP47\+Language\+::preferred\+Value()}}. For example \textquotesingle{}Algerian Saharan Arabic\textquotesingle{} has a prefix of Arabic (ar) and a preferred\+Value() of (aao) and would be specified using \textquotesingle{}ar\+\_\+aao\textquotesingle{}.

\begin{DoxyNote}{Note}
extlang languages have a single prefix, however other types can\# have more than one, VARIANT for example which is why prefix() returns a QVector$<$\+QString$>$ rather than a single value.
\end{DoxyNote}
You can force a data rebuild by calling \mbox{\hyperlink{classBCP47Languages_a9133ce3db4ef8216d96fcf641e105582}{rebuild\+From\+Registry()}};

A different registry file can be set with \mbox{\hyperlink{classBCP47Languages_a14ebc624ae37a6108d30ceb7b347df5f}{set\+Registry()}}

For a more extensive handling see \href{https://www.w3.org/International/articles/language-tags/}{\texttt{ https\+://www.\+w3.\+org/\+International/articles/language-\/tags/}}.

This class loads the language tag definitions from the \href{https://www.iana.org/assignments/language-subtag-registry/language-subtag-registry}{\texttt{ IAIN language tag registry}} and parses it into an easier to use-\/ish form. It will also save the data into a local YAML file for easier/faster recovery.
