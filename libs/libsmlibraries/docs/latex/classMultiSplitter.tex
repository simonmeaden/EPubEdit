\doxysubsection{Detailed Description}
A multidirectional splitter that works in a similar way to the editor panes in Qt\+Creator. 

The \mbox{\hyperlink{classMultiSplitter}{Multi\+Splitter}} is actaully an array of QWidget\textquotesingle{}s and QSplitters. Initially it holds a single QWidget. When the widget is split it replaces the widget with a QSplitter, and places the QWidget and a second QWidget.

Classes that intend to be used in \mbox{\hyperlink{classMultiSplitter}{Multi\+Splitter}} should implement the \mbox{\hyperlink{classCloneableWidgetInterface}{Cloneable\+Widget\+Interface}}. This creates a cloneable widget so that when a widget is \textquotesingle{}split\textquotesingle{} either vertically or horizontally a clone of the forst widget is created and placed into the second split.
