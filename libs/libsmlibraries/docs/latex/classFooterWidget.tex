\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classFooterWidget}{Footer\+Widget}} class defines a \mbox{\hyperlink{classFooterWidget}{Footer\+Widget}} object. 

A \mbox{\hyperlink{classFooterWidget}{Footer\+Widget}} is a QWidget with a header bar in which various widgets can be placed. The actual widgets are stored in a \mbox{\hyperlink{classDockFooter}{Dock\+Footer}} which can be accessed via the \mbox{\hyperlink{classFooterWidget_ad799e823d4e69f511c3af3b1f871b907}{footer()}} method.
