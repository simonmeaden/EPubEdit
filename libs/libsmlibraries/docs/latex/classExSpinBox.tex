\doxysubsection{Detailed Description}
A spin box that will display the value in decimal, hexadecimal, octal or binary.

An extended QSpin\+Box that can display in various common formats, (hexadecimal, octal and binary). \mbox{\hyperlink{classExSpinBox_aa253d1b6202dba708ea336a06c34a9dca005b0dc862993b49b1b34c8467b59cc0}{Ex\+Spin\+Box\+::\+Decimal}} in fact just uses the underlying QSpin\+Box so acts in exactly the same way.

Character case (in the case of hexadecimal) is enforced, uppercase by default, and can be changed using the \mbox{\hyperlink{classExSpinBox_ac99dc6b067e00e4ec440294b501ebfe5}{set\+Char\+Case()}} method to either \mbox{\hyperlink{classExSpinBox_a1d2046fd6c4bb5ac896ed56280954048a3950585d5d45d4f7dd13cdd789ad42c5}{Ex\+Spin\+Box\+::\+Uppercase}} or \mbox{\hyperlink{classExSpinBox_a1d2046fd6c4bb5ac896ed56280954048ade624ec351ddf559f4f43bbf89d692a3}{Ex\+Spin\+Box\+::\+Lowercase}}.

The display type can be set using the \mbox{\hyperlink{classExSpinBox_a582e9029b2f9028647d6f7e7d6afe828}{Ex\+Spin\+Box\+::set\+Display\+Type()}} method to \mbox{\hyperlink{classExSpinBox_aa253d1b6202dba708ea336a06c34a9dca005b0dc862993b49b1b34c8467b59cc0}{Ex\+Spin\+Box\+::\+Decimal}}, \mbox{\hyperlink{classExSpinBox_aa253d1b6202dba708ea336a06c34a9dca8fd7fa451debde033dc2d7e953387a8a}{Ex\+Spin\+Box\+::\+Hexadecimal}}, \mbox{\hyperlink{classExSpinBox_aa253d1b6202dba708ea336a06c34a9dcad67efd040488af0239679f7645a53624}{Ex\+Spin\+Box\+::\+Binary}} or \mbox{\hyperlink{classExSpinBox_aa253d1b6202dba708ea336a06c34a9dcace5500fd19b487c7b78a5ec3ee240e79}{Ex\+Spin\+Box\+::\+Octal}}.

By default leading zeroes are not shown. This can be modified setting the Ex\+Spin\+Box\+::set\+Leading\+Zeroes(bool) to true. The actual number of displayed characters is set when the minimum and maximum range is set.
