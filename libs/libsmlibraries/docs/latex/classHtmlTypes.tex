\doxysubsection{Detailed Description}
A utility class to test for correct html tag names and attribute names. 

Use \mbox{\hyperlink{classHtmlTypes_a784ee336ea9ede17c255bed0c6cf42a2}{is\+Valid\+Tag(\+Tag$\ast$)}} or \mbox{\hyperlink{classHtmlTypes_a6879f6587a7160f0cf94350bf336b909}{is\+Valid\+Tag(const QString\&)}} to test if the tag name is good.

Use \mbox{\hyperlink{classHtmlTypes_a017834f276b8e028b945b48e88390952}{is\+Valid\+Void\+Tag(\+Tag$\ast$)}} or \mbox{\hyperlink{classHtmlTypes_a658ec203a6b7319a535e310f69563ac6}{is\+Valid\+Void\+Tag(const QString\&)}} to test if the tag name is a void tag name good. Void tags are those that are complete in and of themselves for example $<$meta ...$>$. d tags MUST be closed with ../$>$ if the file is xhtml but CAN be closed if the file is an html file.

Use \mbox{\hyperlink{classHtmlTypes_a91d8747007b1c2c6816e78b506353ae8}{is\+Valid\+Attribute\+For\+Tag(const QString\&, const QString\&)}} to check that the attribute is valid for the tag.

Use \mbox{\hyperlink{classHtmlTypes_a0958e34b360118f5c8f786788b4dcf17}{is\+Valid\+Attribute\+Value\+For\+Tag(const QString\&, const QString\&, const QString\&)}} to check that both the attribute and that attribute value is valid for this tag.

\begin{DoxyNote}{Note}
that the checking for attribute value is not exhaustive as many contain text strings that are difficult to check. Values that have fixed values such as 
\begin{DoxyCode}{0}
\DoxyCodeLine{<audio preload>}

\end{DoxyCode}
 which have three valid values (none$\vert$metadata$\vert$auto) are checked correctly but
\begin{DoxyCode}{0}
\DoxyCodeLine{<audio src=\textcolor{stringliteral}{"{}"{}}>}

\end{DoxyCode}
 contains an URI string and is harder to check for correctness only checks that is is a string.

The Tag$\ast$ types are used by \mbox{\hyperlink{classHtmlScanner}{Html\+Scanner}} and Htm\+Highlighter, if you are using it as a standalone just use the QString versions. 
\end{DoxyNote}
