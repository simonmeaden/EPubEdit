\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classDockFooter}{Dock\+Footer}} class is a special case of Dock\+Item that is fixed at the bottom. 

Please note that \mbox{\hyperlink{classDockFooter}{Dock\+Footer}} buttons only allow icons, any text is ignored.
