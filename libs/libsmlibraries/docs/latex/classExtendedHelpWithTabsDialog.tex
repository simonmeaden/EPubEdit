\doxysubsection{Detailed Description}
An help window that displays a more extended help dialog, with a tabbed extension page. 

The dialog displays the help string and holds a \textquotesingle{}Close\textquotesingle{} button, however a \textquotesingle{}More\textquotesingle{} button is also displayed and this will cause a hidden tabbed area, in which more help pages can be stored, to appear. The \textquotesingle{}More\textquotesingle{} button will now read \textquotesingle{}Less\textquotesingle{}.

The \textquotesingle{}Close\textquotesingle{} button returns no value is returned so use the show() method rather that exec() method to invoke the dialog.
