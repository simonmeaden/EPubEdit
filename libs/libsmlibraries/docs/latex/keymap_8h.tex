\doxysubsection{Detailed Description}
stores a set of Key maps for storage in a configuration file. 

\begin{DoxyAuthor}{Author}
Simon Meaden 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2022-\/08-\/10
\end{DoxyDate}
Not really intended for anyone else to use as you can see from to paucity of mapping keys.

Data is stored in a QMap$<$\+Key\+Event\+Mapper, Key\+Mapper$>$ and the main part of the code is to allow the data to be stored/recovered from a YAML file using yaml-\/cpp.
